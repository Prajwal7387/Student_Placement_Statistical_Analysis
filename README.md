# 📊 Student Placement Statistical & Machine Learning Analysis

## 📌 Project Overview
This project analyzes factors affecting student placement using statistical methods and machine learning techniques.

## 📂 Dataset Features
- CGPA
- Internships
- Projects
- Coding Skills
- Communication Skills
- Aptitude Test Score
- Soft Skills Rating
- Certifications
- Backlogs
- Placement Status

## 📊 Statistical Methods Used
- Descriptive Statistics
- T-Test (Hypothesis Testing)
- Confidence Interval Estimation
- Correlation Analysis

## 🤖 Machine Learning Model
- Logistic Regression
- Model Accuracy: XX%

## 🔍 Key Findings
- Higher CGPA increases placement probability.
- Internship experience positively affects placement.
- Coding and aptitude skills strongly influence outcomes.
- Backlogs negatively impact placement chances.
